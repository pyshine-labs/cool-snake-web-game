How to Play
  • Use arrow keys or on‑screen buttons to steer the snake.
  • Eat the red food to grow and earn points.
  • Avoid hitting walls or the snake's own tail.
  • Adjust speed and other settings in the control panel.
  • Customize colors, themes, and visual effects.
About Snake Game

This is a modern take on the classic Snake game, featuring:

  • Glass‑morphism UI with smooth animations
  • Responsive design for all screen sizes
  • Customizable visuals and sound
  • Backend powered by Flask with REST API
  • Real‑time score tracking and high‑score persistence

Created as a demonstration of full‑stack web development using Python, JavaScript, and modern CSS.
